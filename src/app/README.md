# Simply Books

Simply Books is a user-friendly web application designed for book lovers to explore, share, and review their favorite books. This platform provides a seamless experience for discovering new reads and keeping track of your personal library.

## Features

**User Specific**: Takes in a UID so every user's experience is personalized.
**CRUD Functionality**: Full create, read, update, and delete operations for authors and books.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript, React.js, Next.js
- **Backend**: Node.js, Express.js
- **Database**: Firebase
- **Styling**: Bootstrap
- **Tools**: Postman
